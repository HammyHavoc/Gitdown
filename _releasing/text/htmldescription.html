Gitdown

Use this simple plugin to upload and synchronize markdown articles hosted in remote repository to your wordpress site as blog posts.

Set a custom glob pattern to search for posts within your repo and manage them in a little UI.

How to Install

  1. Go to your Wordpress Dashboard.
  2. Go to plugins -> install.
  3. Click on Upload Plugin.
  4. Drag your Zip file into the file input and upload.
  5. Once its uploaded, activate it and your good to go.

You find Help within the main view for the plugin in the help tab. And if you want you can also contact me here info@maximmaeder.com
